# Математика для Data Science — презентации курсов (proglib)

Reveal.js-презентации для коротких видео по 5 курсам.

- Лендинг: `index.html` (5 карточек)
- Курсы: `start/ calculus/ probability/ linear-algebra/ pro/` — у каждого `index.html` (дек) + `script.md` (сценарий озвучки)
- `reveal/` — локальный бандл reveal.js 5.1.0 + тема `proglib.css`
